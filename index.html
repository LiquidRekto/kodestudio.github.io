FEATURES

KodePas has 3 ways for you to develop software
  • NATIVE
  • WEB APP
  • HIBRID

Easy with Pascal

Use Pascal platform to develop. Learning and writing Pascal code is extremely simple

Cross compiler

Your source code will be translated on many different platforms. Write code once - run anywhere

Visual design

The application interface is designed visually based on LFM

High speed

Your applications are fully compatible with the platform on which they operate, so they run extremely fast

Diverse components

There are many components developed for you. Components will help you write code more quickly and conveniently

OOP Code

Everything in the source code is treated as an object, which makes it easy for you to interact with the source code

Power from Pascal and JS

Start using KodePas in the backend and fontend in web. KodePas lock like JS

A lot of components

Similar to playing Minecraft, you just need to put the blocks together. Component that saves you time and effort writing code

Multiple browsers

Your site will be fully compatible on many browsers. Such as: Chrome, FireFox, Opera, Safari,...

Unlimited design

Design your website with HTML and CSS. Not only that, KodePas is also compatible with many design platforms

JS compatible

Double the power with JS. You can use functions created by JS. You can also use KodePas on Node JS

Similar to the Web

Hybrid App is similar to Web App. However, Hybrid has higher speed, interact with deeper hardware, more security,...

Write once, run anywhere

You can use Hybrid App for Desktop and Mobile. All your code will be reused. This saves time, effort and cost for the project.

Diverse components

UI components display beautifully on all mobile devices and platforms. component that gives you deeper interaction with your device's hardware and system.

Code with your favorite Frameworks

We want you to feel most comfortable writing code with KodePas. That is why Hibrid is integrated with many Frameworks such as: Angular, React, Vue, ...

Combination of Native and Web

Easy, fast, simple, beautiful,... are adjectives to talk about KodePas Hibrid
