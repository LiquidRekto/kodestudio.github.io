GitHub

GitHub là nơi Kode dùng để lưu trữ toàn bộ mã nguồn. Từ mã nguồn chính, mã nguồn các Plugin đến các ví dụ.
Ngoài việc lưu trữ thì GitHub cũng là nơi các bạn có thể báo cáo các lỗi thông qua mục Issues.

Nếu bạn truy cập trang GitHub của Kode thì đừng quên tặng chúng tôi một Star nhé!!!

StackOverFlow

Thật thiếu sót nếu chúng ta bỏ qua StackOverFlow - cái động lập trình viên. Nếu bạn gặp khó khăn trong lúc code, đừng ngần ngại đặt câu hỏi trên diễn đàn này. Đừng quên kèm #kode trong câu hỏi nhé
